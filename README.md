# 🇺🇿 Uzbek Names Dataset / O'zbek ismlari to'plami / База узбекских имен

Welcome to the comprehensive open-source dataset of Uzbek names, categorized for developers, data scientists, and creators.

---

## 🇬🇧 English Version

A clean, structured collection of traditional and modern Uzbek names for boys and girls. This project aims to provide a reliable open resource for developers to integrate into their software, applications, or test databases.

### 📦 Repository Structure
The data is separated by gender for ease of use:
* 📄 **`boys`** — A complete list of Uzbek male names.
* 📄 **`girls`** — A complete list of Uzbek female names.
*(Note: You can format these files as `.txt`, `.json`, `.csv`, etc., depending on your repository structure).*

### 🚀 Use Cases
You can freely use this dataset for:
* Generating **dummy data** or mock user profiles for software testing.
* Building **Name Generator** apps, websites, or Telegram bots.
* Form validation, autocomplete features, or data classification.
* Training Artificial Intelligence (AI) and Machine Learning (ML) models.

There is **no copyright** on this dataset. It is released under the **Public Domain**.

## 🇺🇿 O'zbekcha talqin (Uzbek Version)

Dasturchilar, tadqiqotchilar va dizaynerlar o'z dasturiy mahsulotlarida bepul va erkin foydalanishlari uchun jins bo'yicha ajratilgan o'zbekcha ismlarning yirik ro'yxati.

### 📦 Loyiha tarkibi
Fayllar qulaylik yaratish maqsadida ajratilgan:
* 📄 **`boys`** — O'g'il bolalar ismlari ro'yxati.
* 📄 **`girls`** — Qiz bolalar ismlari ro'yxati.

### 🚀 Qayerlarda foydalanish mumkin?
Ushbu ma'lumotlar to'plamidan quyidagi maqsadlarda foydalanish mumkin:
* Dastur va veb-saytlarni sinovdan o'tkazish uchun **"Dummy data" (test ma'lumotlari)** yaratishda.
* Ismlar bilan bog'liq mobil ilovalar, veb-saytlar yoki **Telegram botlar** yaratishda.
* Foydalanuvchilar ro'yxatini avtomatik to'ldirish yoki generatsiya qilishda.
* Ma'lumotlar bazasi va sun'iy intellekt (AI/ML) modellarini o'rgatishda.
